# 音效檔案 (Sound Effects)

請將以下音效檔案放置於此目錄：

| 檔案名稱 | 用途 | 建議風格 |
|---------|------|---------|
| `start.mp3` | 專注協定啟動時播放 | 科幻啟動/電源開啟音效 |
| `warning.mp3` | 違規警告區域時播放 | 雷達嗶聲/警報聲 |
| `penalty.mp3` | 懲罰執行時播放 | 故障/失敗音效 |
| `complete.mp3` | 任務完成時播放 | 成功/解鎖音效 |

## 推薦音效來源

- [Freesound.org](https://freesound.org/) - 免費音效庫
- [Pixabay](https://pixabay.com/sound-effects/) - 免費音效
- [Zapsplat](https://www.zapsplat.com/) - 免費音效（需註冊）

## 音效風格建議

為了符合賽博龐克主題，建議選擇：
- 電子/合成器風格
- 科幻/未來感
- 故障藝術 (Glitch) 風格
